# validacion-correo-php

Esta clases creada en php, sirve para la validación de una dirección de correo electrónico.

Recibe como argumento la dirección de correo electrónico en un string.

Devuelve una variable tipo boolean que define si la dirección de correo es válida o no.

Se puede implementar en cualquier formulario que tenga campos con dirección de correo electrónico.
